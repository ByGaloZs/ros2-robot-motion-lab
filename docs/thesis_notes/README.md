# Thesis Notes

This folder contains rough working notes that may later be transformed into sections of
the Master's Thesis / TFM.

Do not treat these files as final thesis prose. Use them to collect technical evidence,
links to experiments, architecture decisions, and implementation notes.

## Chapter Mapping

- `tfm-thesis-definition.md`: canonical working thesis, hypothesis, objectives, scope,
  and target architecture vision.
- `master_subject_alignment.md`: mapping between the UC3M master's subjects, selected
  electives, and the TFM direction.
- `chapter_01_introduction_notes.md`: problem, motivation, project context.
- `chapter_03_methodology_notes.md`: development process, experiments, validation
  method.
- `chapter_04_architecture_notes.md`: modular architecture and design decisions.
- `chapter_05_experiments_notes.md`: experiment results and technical validation.
- `learning_map.md`: study path and technical learning map.
